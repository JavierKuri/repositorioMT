Prácticas desarrolladas durante el semestre

Nombre Enlaces ¿Qué hicimos? ¿Qué aprendimos?
Práctica 2. HTML Práctica 2. HTML Realizamos un archivo con puro HTML (temática libre) utilizando diversos tipos de etiquetas (form, input, h1-h6, listas ordenadas y descriptivas). Logramos ser mas familiares con el funcionamiento de las etiquetas en HTML, entendiendo su estructura general y sus jerarquias de importancia y posicionamiento.
Práctica 3. Tablas en HTML, enlaces y CSS Práctica 3. Tablas en HTML, enlaces y CSS Creamos tres archivos con tablas diferentes (con estilos diferentes de CSS), a los cuales se obtiene acceso mediante una página principal. Aprendimos a utilizar enlaces para movernos entre archivo y a aplicar estilos básicos de CSS a nuestros proyectos, ya sea desde archivos externos o mediante la etoqueta style internamente.
Práctica 4. Javascript 1
  • Transformador de calificaciones
  • Función factorial recursiva
  • Formulario con JS
 Creamos tres archivos de HTML con JavaScript, de tal forma que cada uno realizaba una función diferente. Aprendimos el funcionamiento básico del control de flujo y funciones en JavaScript, utilizando la etiqueta script dentro de cada archivo para lograr los objetivos específicos.
Práctica 5. JavaScript 2 Práctica 5. JavaScript 2 Creamos un formulario que fuera llenando una tabla con cada entrada que se registraba. Aprendimos a crear "tablas dinámicas" mediante JS, recalcando la importancia que tiene el atributo id para la extracción y modificacion de los datos del archivo HTML.
Lista de Tareas Lista de Tareas Como parte de la evaluación de medio término creamos un archivo con HTML y JS para crear una lista de tareas, en donde se tachen las completadas. Aprendimos a crear tablas dinámicas con elementos de mayor funcionalidad y a utilizar otras etiquetas (span) como apoyo para cumplir el objetivo de tachar los textos de las tareas completadas.
